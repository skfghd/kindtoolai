📋 KINDTOOL.AI 헤더 - 복사 & 붙여넣기

각 앱에 아래 코드를 복사해서 붙여넣기만 하면 됩니다!

🚀 단 한 줄로 완성!

아래 코드를 복사해서 각 외부 앱의 HTML 파일에 붙여넣으면 자동으로 KINDTOOL.AI 헤더가 추가됩니다.

✅ 이 코드의 장점:
  • 기존 디자인을 전혀 건드리지 않음
  • 모바일에서도 자동으로 반응형 동작
  • 구글 애널리틱스 추적 자동 포함
  • 중복 설치 방지 기능 내장
🔗 공통 코드 (모든 앱에 동일하게 사용)
<script src="https://kindtool.ai/kindtool-header.js" async></script>

추가 위치: HTML 파일의 <head> 태그 안이나 <body> 태그 끝부분
