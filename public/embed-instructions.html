🚀 KINDTOOL.AI 공통 헤더 삽입 가이드

📋 목적: 모든 외부 도구 앱들의 최상단에 통일된 KINDTOOL.AI 메뉴바를 추가하여 사용자 경험을 개선합니다.

1. 📁 스크립트 파일 위치

공통 헤더 스크립트는 다음 URL에서 사용할 수 있습니다:

스크립트 URL:
https://kindtool.ai/kindtool-header.js

2. 🔧 각 외부 앱에 추가하는 방법

방법 1: HTML에 직접 추가 (권장)

각 외부 앱의 <head> 태그 또는 <body> 태그 맨 마지막에 다음 코드를 추가하세요:

<script src="https://kindtool.ai/kindtool-header.js" async></script>

방법 2: JavaScript로 동적 로딩

기존 JavaScript 코드에서 동적으로 로드하려면:

const script = document.createElement('script'); script.src = 'https://kindtool.ai/kindtool-header.js'; script.async = true; document.head.appendChild(script);

6. 🎨 디자인 특징

  • 비침입적: 기존 앱 디자인에 영향을 주지 않음
  • 고정 위치: 페이지 최상단에 고정된 헤더
  • 반투명 배경: 기존 콘텐츠와 자연스럽게 조화
  • 자동 여백: 기존 콘텐츠를 64px 아래로 자동 이동
  • Z-index 최고값: 다른 요소들 위에 항상 표시
⚠️ 주의사항:
  • 스크립트는 중복 로드를 방지하는 로직이 포함되어 있습니다
  • 기존 페이지의 CSS와 충돌하지 않도록 네임스페이스를 사용합니다
  • 모든 스타일은 !important 규칙으로 보호됩니다

7. 🧪 테스트 방법

  1. 각 앱에 스크립트를 추가합니다
  2. 페이지를 새로고침합니다
  3. 상단에 KINDTOOL.AI 헤더가 나타나는지 확인합니다
  4. 모든 링크가 올바르게 작동하는지 테스트합니다
  5. 모바일에서도 정상 작동하는지 확인합니다

8. 🔄 업데이트 방법

헤더 디자인이나 기능을 수정해야 할 경우:

  1. kindtool-header.js 파일을 수정합니다
  2. 모든 외부 앱에서 자동으로 업데이트된 헤더를 사용하게 됩니다
  3. 브라우저 캐시 때문에 즉시 반영되지 않을 수 있으니 강제 새로고침하세요
💡 팁: 각 앱의 개발자 도구(F12)에서 콘솔을 확인하여 스크립트가 정상적으로 로드되었는지 확인할 수 있습니다.
