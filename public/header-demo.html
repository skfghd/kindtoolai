📌 위쪽을 확인해보세요! KINDTOOL.AI 공통 헤더가 자동으로 추가되었습니다.

✨ 제공되는 기능들

🏠 KINDTOOL.AI 메인 사이트로 바로 이동
⚡ 모든 도구들에 대한 드롭다운 메뉴
📱 모바일 친화적인 반응형 디자인
🎨 기존 앱 디자인에 영향을 주지 않는 비침입적 구현
📊 구글 애널리틱스 이벤트 추적
⚡ 빠른 로딩과 캐싱 최적화

🔧 사용 방법

각 외부 앱에 다음 한 줄만 추가하면 됩니다:

<script src="https://kindtool.ai/kindtool-header.js" async></script>
📖 상세 가이드 보기 🏠 메인 사이트로

이 데모 페이지는 실제 외부 앱에서 헤더가 어떻게 보이는지 시연하기 위한 것입니다.

실제 외부 앱들에 적용하면 동일한 헤더가 나타납니다.
